## INXI


# Most useful inxi commands for Linux

*   `inxi -F` : Full output, provides a comprehensive overview of your system.
*   `inxi -Fxz` : Full output, with 'x' for extra details and 'z' to hide sensitive information like MAC addresses.
*   `inxi -S` : Show system information (Host, Kernel, Desktop Environment, etc.).
*   `inxi -C` : Show CPU information.
*   `inxi -G` : Show graphics card and display information.
*   `inxi -M` : Show motherboard information.
*   `inxi -D` : Show hard disk drive information.
*   `inxi -N` : Show network card information.
*   `inxi -A` : Show audio card information.
*   `inxi -R` : Show RAID controller information.
*   `inxi -B` : Show battery information (if applicable).
*   `inxi -I` : Show information about running processes (init, runlevel).
*   `inxi -t cm` : Show top 10 CPU and memory processes.
*   `inxi -v8` : Very verbose mode, useful for detailed troubleshooting.
